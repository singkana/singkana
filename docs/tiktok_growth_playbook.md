# TikTok Growth Playbook（SingKANA）

## 目的
TikTok流入 → LP → 変換 → 課金（trial/subscribe）までのファネルを可視化し、「勝ち型（勝ちフック）」を高速に量産・反復する。

---

## 7.1 投稿テンプレ（そのまま撮れる）

### テンプレA：Before/After固定フォーマット（8秒）
- **0–1秒**: 冒頭フック（固定文言）
  - 「通常のローマ字 vs SingKANA（歌唱最適化）」  
- **1–6秒**: 画面にBefore/Afterを並べる（固定例でもOK）
  - 上：普通のローマ字
  - 下：歌える表記（伸ばし/促音/息継ぎ/連結が入ってる）
- **6–8秒**: CTA
  - 「リンクから一発変換してみて」 / 「無料で試せる」

### テンプレB：コメント返信型（最強）
- **画面**: 「返信」UIを入れてコメントを見せる
- **構成**
  - 「その歌詞、歌える形にするとこう」→ Before/After
  - 最後に「他にも曲名/一節コメントして」

### テンプレC：制作工程短縮（倍速）
- **見せるもの**
  - 変換 → コピー → SRT/TXT → すぐ歌える（台本化）
- **狙い**
  - 「制作が速い」= クリエイター訴求（“自分にもできそう”）

---

## 7.2 1日運用（15本/日で回る）
- **体制**: 5アカ × 1日3本
- **週2**: 当たり動画をSpark候補に指定

### 見る指標（TikTok + サイト）
- **TikTok側**
  - 3秒維持率
  - 平均視聴維持率
  - プロフ遷移率
  - LPクリック率
- **サイト側（ファネル）**
  - `convert_success / landing_click`
  - `trial_start / convert_success`

---

## 7.3 40本で勝ちフック確定ルール
- **40本投稿したら** 上位10本の共通点を抽出
- **共通点が2個以下** → フックが弱い → フックを作り直す
- **共通点が3個以上** → 勝ち型確定 → その型だけを擦る
